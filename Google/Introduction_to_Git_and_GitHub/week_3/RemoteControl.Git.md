### Работа с пультами

```
# Посмотреть конфигурацию удаленного репозитория
git remote -v
# Подробная конфигурация удаленного репозитория
git remote show -origin
# Просмотреть удаленные ветки репозитория
git branch -r
# Синхронизировать удаленный репозиторий с локальным (обновить фиксации других участников локально)
git fetch
# Просмотреть коммиты других участников в удаленном репозитории
git log <name...branch>
# Объеденить удаленную и локальную ветки
# git merge <name...branch>
```
